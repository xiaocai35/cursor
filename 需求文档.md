# 聚贤人力 - 蓝领招聘平台需求文档

## 1. 项目概述

### 1.1 项目背景
聚贤人力是一家专注劳务派遣20年的福建省泉州地区人力资源服务领军企业，拥有30+合作工厂、5000+人才储备。本项目旨在构建一个专为蓝领工人设计的移动端求职招聘平台，提供完整的求职、入职、薪资管理和企业服务功能。

### 1.2 项目目标
- 为蓝领工人提供便捷的求职招聘服务
- 为企业提供高效的人力资源管理工具
- 建立完善的工资预支和财务管理系统
- 打造移动端优先的用户体验

### 1.3 目标用户
- **主要用户**: 蓝领工人（求职者）
- **次要用户**: 企业HR、工厂管理人员
- **管理用户**: 聚贤人力内部员工

## 2. 技术架构

### 2.1 技术栈
- **前端框架**: HTML5 + Tailwind CSS
- **脚本语言**: JavaScript ES6+
- **数据存储**: LocalStorage（本地持久化）
- **设计规范**: 移动端优先（414px最大宽度）
- **主题色彩**: 紫色系（#8A33D1）

### 2.2 兼容性要求
- 移动端：iOS 10+, Android 7+
- 微信小程序内嵌浏览器
- 主流移动浏览器（Chrome, Safari, UC等）

    - 安全退出：带确认对话框的退出功能

  - 分组设计：按照功能类型将设置项分为6大模块
  - 图标色彩系统：不同功能模块使用不同颜色的图标标识

#### 3.1.5 财务管理模块

**3.1.5.1 工资预支系统** (`advance-salary.html`)
- **功能描述**: 工资预支申请和管理
- **核心特性**:
  - 用户状态展示（姓名、职位、工作天数）
  - 智能预支资格检查
  - 三种状态处理：
    - ✅ 符合条件：显示300元预支额度
    - ⏳ 未满7天：显示剩余天数
    - 🚫 已申请：显示下月可用时间

  - 预支历史记录展示
- **业务规则**:
  - 入职满7天后可申请
  - 每次固定300元
  - 每月只能申请一次
  - 状态管理：申请 → 审核 → 发放 → 完成

## 6. 业务流程

### 6.1 求职流程
1. 用户注册/登录
2. 浏览职位列表
3. 查看职位详情
4. 完善个人信息
5. 实名认证
6. 提交职位申请
7. 选择集合门店
8. 获得录用通知
9. 入职报到

### 6.2 工资预支流程
1. 检查资格条件（入职满7天）
2. 确认预支金额（300元）
3. 提交申请
4. 等待审核
5. 预支发放
6. 从工资中扣除

### 6.3 提现流程
1. 选择提现金额
2. 选择提现方式
3. 确认提现信息
4. 提交申请
5. 处理中状态
6. 提现完成

### 6.4 反馈流程
1. 选择反馈类型（投诉/建议）
2. 选择问题分类
3. 填写问题描述
4. 上传相关图片（可选）
5. 提交反馈
6. 查看处理状态
7. 收到处理结果

## 7. 安全性要求

### 7.1 数据安全
- 敏感信息本地加密存储
- 身份证号码脱敏显示
- 手机号码脱敏显示
- 用户设置数据加密保存

### 7.2 业务安全
- 工资预支资格严格验证
- 实名认证必须完成
- 提现金额限制和验证
- 设置修改需要验证身份

## 8. 性能要求

### 8.1 页面性能
- 页面加载时间 < 2秒
- 图片懒加载
- 代码压缩优化
- 设置状态快速加载和保存

### 8.2 用户体验
- 流畅的页面切换动画
- 及时的用户反馈
- 离线功能支持
- 设置界面响应速度 < 300ms
